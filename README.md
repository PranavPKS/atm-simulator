# 1. How to start

```shell
$ git clone https://github.com/shiprabehera/atm-simulator.git
$ cd atm-simulator
$ mvn spring-boot:run
$ Open localhost:8080 on browser
$ Open localhost:8080/h2-console to login to the database:
  1. JDBC URL: jdbc:h2:mem:testdb
  2. Username: sa
  3. Password: password

```
